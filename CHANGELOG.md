# Changelog

## [1.1.0] - 2024-12-10

### Added
- **SVG icon support** - Use SVG files directly with react-native-svg-transformer
- New icon type: `type: 'svg'` with component prop
- Auto-color for SVG icons based on active state
- Support for custom vector files

### Requirements
- Optional: react-native-svg + react-native-svg-transformer for SVG support

## [1.0.0] - 2024-12-10

### First Stable Release
- Item-agnostic design
- Glassy transparent background
- Vector icons (5 families)
- PNG/Image icons
- RTL/LTR support
- Visible/disabled props
- Full TypeScript support
